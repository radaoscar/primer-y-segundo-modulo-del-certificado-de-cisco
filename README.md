# primer-y-segundo-modulo-del-certificado-de-cisco

Resumen modulo 1: comunicación en un mundo
conectado

Tipos de red:

Internet es una red mundial de redes interconectadas que permite el
intercambio de información mediante estándares comunes. Existen redes
domésticas pequeñas (SOHO), redes medianas y grandes como las
corporativas, y una gran variedad de dispositivos conectados, como celulares,
televisores inteligentes, sensores y automóviles.


Transmisión de datos:

Los datos personales se clasifican en voluntarios (compartidos por el
usuario), observados (registrados por acciones) e inferidos (deducidos del
análisis). El bit es la unidad mínima de información. Los datos se transmiten
mediante señales eléctricas, ópticas o inalámbricas.


Ancho de banda y rendimiento:

El ancho de banda es la cantidad de datos que pueden transmitirse en un
tiempo determinado y se mide en Kbps, Mbps y Gbps. El rendimiento real
puede variar debido al volumen de datos y la latencia, que es el tiempo que
tardan los datos en viajar de un punto a otro.

Resumen modulo 2: componentes, tipos y conexiones de
red


Clientes y Servidores:

Las computadoras conectadas a una red se llaman hosts y pueden funcionar
como clientes, servidores o ambos, según el software que utilicen. En redes
pequeñas se usa el modelo P2P, donde las computadoras comparten
recursos entre sí. En redes grandes se emplean servidores dedicados, que
ofrecen mayor rendimiento, seguridad y escalabilidad.
